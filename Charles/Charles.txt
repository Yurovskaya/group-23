Charles Проксирование трафика на компьютере

Заработал трафик

1. Вкладка Proxy
2. Включить Windows proxy

По умолчанию работают с http трафиком.
Чтобы сниффер расшифровывал https трафик, необходимо на компьютере поставить сертификат. 

1. https://www.charlesproxy.com/documentation/using-charles/ssl-certificates/
2.  "SSL Proxying > Install Charles Root Certificate"
3. Зайти  - Help
          - SSL Proxyng
          - Install Charles Root Certificate
          - Установить сертификат
          - Текущий пользователь ( Локальный копьютер( сделать те же шаги))
          - Поместить все сертификаты в следующее хранилище
          - Обзор
          - Доверенные корневые центры сертификации
          - Готово 
4. Локальный копьютер( сделать те же шаги)
5. - Проверить в Параметрах сети и интернет
   - Прокси-сервер
   - Использовать прокси-сервер (ползунок включен)
   - Порт 8888
6. - Proxy
   - SSL proxying settings
   - Enable SSL Prixying (поставить галочку)
   - Add
Ввести в поле хост
   - *.*
   - []
   - *

Проксирование трафика на телефоне
Все должно быть в одной wi-fi сети
1. - Help
   - SSL proxying Install Charles Root Certificate on Mobile Device or Remote Browser
   - Заходим в телефоне в настройки wi-fi
   - Жмем на имя сети
   - Настройка прокси
   - Вручную
   - сервер прописываем 172.20.10.4 ( то что вернул Charls)
   - порт 8888
   - Alow

Ставим сертификат на телефон и доверимся ему
1.  - Help
    - SSL proxying Install Charles Root Certificate on Mobile Device or Remote Browser
    - Открываем браузер на телефоне и прописываем то что вернул Charls http://chls.pro/ssl
    - Ждмем разрешить
    - Настройки телефона
    - Основные
    - Профиль
    - Выбираем загруженный профиль
    - Установить

Довериться прокси
- Настройки телефона
- Основные
- Об этом устройстве
- Доверие сертификатам
-  Charles Proxy CA Выбираем ползунок
